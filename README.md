# AI/ML-Based Pharmacy Management System

This is a full-stack web application built using Django that helps manage pharmacy operations and allows users to order medicines online.

## Features

* Medicine inventory management
* Online medicine ordering
* Prescription upload and approval
* Role-based access (Super Admin, Admin, Customer)
* AI features:

  * Symptom-based medicine recommendation
  * Smart search (auto-suggestions)
  * OCR for prescription
  * Demand prediction

## Tech Stack

* Backend: Django, Django REST Framework
* Frontend: HTML, CSS, JavaScript
* Database: SQLite/PostgreSQL
* AI/ML: Python (scikit-learn, OCR)

## Installation

1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run server:
   python manage.py runserver
